WordPressからnote.comへのリンクを簡単にカスタマイズできるプラグイン、デザインや表示するテキストを変えて簡単に連携する事ができます。

== Description ==

Link With Noteはワードプレスからnote.comへのリンクを簡単にカスタマイズしたいという要望から生まれた無料プラグインです。

通常はワードプレスからnoteへのリンクを貼るには、note側の記事から「サイトに貼る」コードをコピーしてワードプレスの記事に貼り付ける事で記事をリンクすると思いますが、このコードはembedタグが使用されているためnoteの情報を増やしたりデザインを変更するには不便な構造になっています。

そこでLink With Noteはnote記事のURLを元にnote APIから記事とクリエイターに関する情報を取得し、オリジナルのnoteへのリンクを生成します。
